# c-send 使い方

この文書は、`c-send` を初めて使う人向けの操作手順をまとめたものです。

## まずやること

1. `csend.exe` を任意のフォルダに置きます。
2. `csend.exe` を起動します。
3. タスクトレイのアイコンから本体画面を開きます。

`setting.ini` がない場合は、起動時に同じフォルダへ作成されます。

## ふつうの使い方

1. 画面上部のカテゴリを選びます。
2. 一覧から送りたい定型文をクリックします。
3. クリップボードにコピーされます。

コピー時は、カーソル付近にアプリ内 tip が表示されます。

## カテゴリの作成

カテゴリは、定型文をまとめるための単位です。

### 手順

1. カテゴリ設定画面を開きます。
2. 一覧のいちばん下にある `（追加）` を選びます。
3. 右側の入力欄に、カテゴリ名と参照先を入力します。
4. `SAVE` を押します。

### 入力する内容

- カテゴリ名
- データファイルのパス、または URL

### 参照先の例

- `samples\jp\Git.txt`
- `C:\Users\example\Documents\my-category.txt`
- `https://example.com/category.txt`

## カテゴリ作成で迷いやすいところ

- `（追加）` を選ぶと、新規カテゴリとして編集できます。
- 名前か参照先が空のままだと保存されません。
- 同じカテゴリ名、同じ参照先は登録できません。
- URL を指定した場合は、保存前に接続確認が行われます。
- URL カテゴリは参照専用なので、一覧の中身は編集できません。

## 並べ替えと削除

- `up` と `down` でカテゴリの順番を変えられます。
- `Delete` でカテゴリを削除できます。
- 削除してもデータファイル自体は消えません。

## よくある見え方

- `URL` と付いたカテゴリは URL 参照です。
- `RO` と付いたカテゴリは読み取り専用です。

## 補足

- カテゴリの設定は `setting.ini` の `[category]` に保存されます。
- 詳しいキー仕様は [setting.ini 仕様](./setting-ini-spec.md) を参照してください。
